# Python Flask Assignment

## In this repo you will learn:

- How to make a virtual environment
- How to extract dependencies in a virtualenv
- How to make a simple Flask app
- How to receive data from client to the Flask server
- How to process data from client and return the value back
- How to use jinja templating

## Instructions

1. Fork this repository
2. Clone the repository you forked.
3. Open one of the folders.
4. Read the `README.md` file inside the folder and follow the instructions.
